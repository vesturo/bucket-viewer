# S3 Bucket Viewer

Static GitHub Pages viewer for public S3-compatible bucket listings.

## GitHub Pages

Publish the repository from the default branch root. The app entrypoint is `index.html`; no Node server or build step is required.

The bucket preset dropdown defaults to Somnium and also includes `ai-space`. The local `download.xml` snapshot is only used as a fallback for the `ai-space` preset.
